项目状态报告
================================================================================

项目名称: Desk_Pet (樱小宠 - 桌面宠物)

项目类型: Python GUI 应用程序

技术栈
--------------------------------------------------------------------------------
- PyQt6: GUI 框架
- 豆包 AI (Doubao): 视频内容总结服务
- urllib/requests: HTTP 请求处理
- JSON: 数据持久化存储
- 高德地图 Web API: 珞珈风貌地图展示

功能模块
--------------------------------------------------------------------------------
1. 桌面宠物显示
   - 无边框窗口
   - 窗口置顶
   - 透明背景
   - 尺寸随等级变化 (118px ~ 156px)
   - 等级色彩渲染效果
   - 显示静态图片

2. 成长等级系统 (5级)
   - Lv1 初来乍到: 0 EXP, 118px, #9FB3D9
   - Lv2 有点期待: 30 EXP, 128px, #7DC2FF
   - Lv3 热爱上头: 80 EXP, 138px, #77E1B2
   - Lv4 角落守护者: 150 EXP, 146px, #FFD27D
   - Lv5 未来引路人: 240 EXP, 156px, #FF9CB9

3. 交互功能
   - 左键拖拽移动
   - 右键弹出上下文菜单

4. 右键菜单项
   - 查看成长面板: 显示当前等级、经验值、连续打卡天数
   - 记录被忽视角落 (+10 EXP): 记录武大校园角落
   - 热爱打卡 (+15起 EXP): 记录热爱行动，支持投入时长奖励
   - 生成三年后寄语: AI 生成未来自我寄语
   - 打开成长主页: 在浏览器打开成长数据页面
   - 输入抖音链接: 保存链接到本地
   - 生成视频总结: 解析链接并生成 AI 总结
   - 查看已保存链接: 列表展示，支持打开链接和删除
   - 珞珈风貌: 打开高德地图展示武大校园
   - 退出: 关闭应用

5. 打卡系统
   - 连续打卡奖励: 每3天 +5 EXP
   - 中断惩罚: 超过1天未打卡 -5 EXP，重置连击
   - 热爱打卡动态奖励: 每30分钟 +2 EXP，上限 +10

6. 链接处理流程
   - 短链还原: 访问短链获取最终 URL
   - 视频 ID 提取: 从 URL 解析视频 ID
   - 内容提取: 多策略提取视频文本信息

7. 视频总结生成
   - 调用豆包 AI (doubao-seed-2.0-lite 模型)
   - 输出格式: 关键要点 + 简短摘要 + 关键词

8. 珞珈风貌地图
   - 高德地图 Web API 集成
   - 显示武大校园位置信息

运行环境要求
--------------------------------------------------------------------------------
- Python 3.x
- PyQt6
- 必须配置环境变量:
  - DOUBAO_API_KEY (豆包 AI)
  - AMAP_WEB_API_KEY (高德地图 Web API)
- 可选配置:
  - DOUBAO_MODEL (默认: doubao-seed-2.0-lite)
  - DOUBAO_BASE_URL (默认: https://ark.cn-beijing.volces.com/api/v3)

数据存储格式
--------------------------------------------------------------------------------
douyin_links.json:
  - raw_text: 原始分享文案 (部分记录)
  - url: 规范化抖音链接
  - created_at: 保存时间 (ISO 格式)

video_summaries.json:
  - raw_input: 用户原始输入
  - short_url: 短链
  - resolved_url: 还原后的完整 URL
  - extracted_text: 提取的文本信息
  - summary: AI 生成的总结内容
  - created_at: 生成时间 (ISO 格式)

pet_growth.json:
  - exp: 当前经验值
  - streak_days: 连续打卡天数
  - last_active_date: 最后活跃日期
  - corner_logs[]: 角落记录列表 (最多80条)
  - passion_tasks[]: 热爱打卡列表 (最多120条)
  - future_messages[]: 未来寄语列表 (最多50条)
